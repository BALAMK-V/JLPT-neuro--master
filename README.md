# JLPT-neuro--master
JLPT Neuro Master Smart JLPT Japanese Learning System (N5–N1) A spaced-repetition-based study platform for mastering Japanese: kanji, vocabulary, listening, reading comprehension, and grammar. Three learning-style modes (Balanced / Focus Support / Calm Structure) Tech Stack: Django + DRF + PostgreSQL (backend) · React + Vite (frontend) · Docker
